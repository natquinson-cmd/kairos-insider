## "Insider trading : tout ce qu'on t'a dit est faux"

**1/11**
```
95% des "insider trading" aux US sont 100% LÉGAUX.

```

**2/11**
```
Distinction cruciale :

✅ INSIDER TRADING LÉGAL (95%) :
→ Dirigeant achète/vend ses propres actions
→ Pas d'info non-publique utilisée
→ Déclare à la SEC sous 2 jours (Form 4)

❌ DÉLIT D'INITIÉ (5%) :
→ Utilise infos confidentielles
→ Pas déclaré / retardé

Les 95% sont PUBLICS. Et exploitables.
```

**3/11**
```
Qu'est-ce qu'un "insider" au sens SEC ?

(Section 16 du Securities Exchange Act)

→ Détient > 10% d'une société cotée
→ OU est dirigeant (CEO, CFO, COO...)
→ OU est administrateur du board

Ces personnes SONT obligées de déclarer CHAQUE transaction sur leurs actions.

Sanction si oubli : amende + perte de mandat.
```

**4/11**
```
Le Form 4, c'est quoi concrètement ?

Un document 1-page, déposé à la SEC.

Champs clés :
1️⃣ Code transaction (P=achat, S=vente, A=award...)
2️⃣ Nombre d'actions
3️⃣ Prix unitaire
4️⃣ Position totale après transaction

Délai : **2 jours ouvrés** après la transaction.

Gratuit sur sec.gov/cgi-bin/browse-edgar
```

**5/11**
```
Les 15 codes de transaction à connaître :

Code P (Purchase, achat marché) → 🟢 LE signal à surveiller
Code S (Sale, vente marché) → 🟡 ambigu (impôts, diversif…)
Code A (Award gratuit) → ⚪ neutre
Code M (Exercise options) → ⚪ neutre
Code F (Tax withholding) → ⚪ neutre

→ 80% des tweets "insider achète !" concernent des codes autres que P. VRAI signal = P.
```

**6/11**
```
Pourquoi les ACHATS valent 10× les VENTES :

Peter Lynch (One Up On Wall Street) :
"Les insiders peuvent vendre pour 1000 raisons personnelles. Ils n'achètent que pour UNE : ils pensent que le cours va monter."

Étude Seyhun (Michigan, 2000) :
→ Achats : +5% sur 6 mois vs marché
→ Ventes : 0% d'alpha stat significant
```

**7/11**
```
Le signal ULTIME : le "cluster" insider.

Définition : ≥ 3 dirigeants d'une MÊME société achètent en 7-30 jours.

Cohen, Malloy & Pomorski (Harvard, 2012, Journal of Finance vol 67) :
→ Cluster + rôles diversifiés + opportuniste = +15% sur 6 mois

3 achats isolés = chance
3 achats coordonnés = quelqu'un sait quelque chose
```

**8/11**
```
Les 5 études académiques qui valident l'alpha des insiders :

📚 Seyhun 1998, 2000 — +5% achats, +11% clusters
📚 Lakonishok & Lee 2001 — +4,8%/an sur 12 mois
📚 Jeng, Metrick, Zeckhauser 2003 — 11,7% alpha long-short
📚 Cohen-Malloy-Pomorski 2012 — +12% opportunistic insiders
📚 Ali & Hirshleifer 2017 — confirmation +10-15%/an

Littérature unanime.
```

**9/11**
```
Les FAUX POSITIFS à éviter :

❌ CEO achète 10k$ alors qu'il gagne 5M$/an → symbolique
❌ Achat 10b5-1 (plan programmé) → neutre
❌ Cluster sur penny stock < 500M$ → risque pump
❌ CEO sortant (retraite) qui "signale" → pas fiable

RÈGLE D'OR : achat > 10% de la compensation annuelle de l'insider = conviction vraie.
```
