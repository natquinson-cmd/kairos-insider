# Kairos Insider

> **Voyez ce que les pros voient.**
> La première plateforme francophone dédiée au Smart Money — suivez les mouvements des insiders, hedge funds et grands investisseurs, en toute légalité.

🌐 **Site** : [kairosinsider.fr](https://kairosinsider.fr)

## Concept

Kairos Insider agrège et traduit en français les déclarations officielles des régulateurs financiers (SEC, AMF, FINRA, CFTC, BaFin, ESMA) pour révéler ce que font les grands investisseurs — en temps quasi réel et à la portée de tous.

## Sources de données publiques

- **SEC Form 4** — Transactions d'insiders US
- **SEC Form 13F** — Portefeuilles des hedge funds
- **AMF France** — Franchissements de seuils
- **COT Report (CFTC)** — Positions institutionnelles
- **FINRA** — Short interest & dark pools
- **BaFin & ESMA** — Data européenne

## Stack technique

- Frontend : HTML / CSS / JS vanilla
- Base de données : Firebase Realtime Database
- Hébergement : GitHub Pages
- Domaine : kairosinsider.fr
